FONTLOG for the struggle font

This file provides detailed information on the struggle Font Software.
This information should be distributed along with the struggle fonts
and any derivative works.

Basic Font Information

struggle is a Unicode typeface family that supports all languages that
use the Latin script and its variants, and could be expanded to support other
scripts.
